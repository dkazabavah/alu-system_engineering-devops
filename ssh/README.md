# SSH - System Engineering DevOps

This directory contains tasks related to SSH configuration and secure remote access.

## Project Tasks

0-use_a_private_key  
A Bash script that uses the private key `~/.ssh/school` to connect to a server.

1-create_ssh_key_pair  
A Bash script that creates an RSA key pair named `school` with the passphrase `betty`.

2-ssh_config  
SSH configuration file that configures the client to use the private key `~/.ssh/school` and disables password authentication.

## Requirements

- Ubuntu 20.04 LTS
- Bash scripts
- SSH client
